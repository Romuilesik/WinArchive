English - English

Please make sure to read README.md and Readme.txt.

To run the program, you need to:

1. Download the Icons folder (for creating a Windows registry key).
2. Download the Sounds folder (the program won't start without sound files).
3. Download Delete_registry_key.exe (for deleting the registry key if something goes wrong).
4. Download WinArchive.exe (the main program).

For easier downloading and additional files:

1. Download WinArchive.rar.
2. Unpack the WinArchive.rar archive into any folder.
