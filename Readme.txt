English - Englsih

Whenever you run the program for the first time or intend to delete a registry key, 
make sure to run the program as an administrator. 
This is necessary for the program to configure icons and files for usage.



Russian - Русский

Всегда, когда запускаете программу впервые или планируете удалить ключ из реестра, 
обязательно запускайте программу от имени администратора. 
Это необходимо для того, чтобы программа могла настроить иконки и файлы для использования.



Ukrainian - Українська


Коли ви запускаєте програму вперше або плануєте видалити ключ з реєстру, 
обов'язково запускайте програму від імені адміністратора. 
Це необхідно для того, щоб програма могла налаштувати піктограми та файли для використання.
