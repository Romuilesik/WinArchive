Для работы программы нужно:

1. Скачать папку Icons (для создания ключа реестра Windows).
2. Скачать папку Sounds (Без звуков программа не запуститься).
3. Скачать Delete_registry_key.exe (для удаления ключа реестра (если пойдет что-то не так)).
4. Скачать WinArchive.exe (основная программа).

Для упрощения скачивания и дополнительные файлы:

1. Скачать WinArchive.rar.
2. Разархивируйте архив WinArchive.rar в любую папку.
